OS Kernel Architecture

x86-64 hobby kernel — ~38K LOC, 60 subsystems, Multiboot1 boot

Boot Sequence

  • • Multiboot1 loads kernel at 0x100000
  • • _start (32-bit): set up identity-mapped page tables
  • • Enable PAE → long mode → paging
  • • kernel_main: 29-step linear init
  • • Interrupts on → idle loop

Key Features

  • • 4-level paging, PHYS_TO_VIRT addressing
  • • Identity map removed (pure high-half VMA)
  • • Kernel stack guard pages
  • • Copy-on-Write fork + per-process PML4
  • • SMP: multi-CPU via ACPI MADT

Testing & CI

  • • E2E telnet-driven shell tests
  • • virtio-net + USB/FAT smoke tests
  • • Host-side libc unit tests
  • • cppcheck, -Werror, ccache
